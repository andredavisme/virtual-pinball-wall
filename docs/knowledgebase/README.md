# Knowledgebase

This directory stores research, references, architecture decisions, and design notes for the Virtual Pinball Wall project.

## Structure

```
docs/knowledgebase/
├── decisions-log.md        # Architecture and design decisions with rationale
├── research/               # External references, hardware specs, framework comparisons
├── architecture/           # System design diagrams and notes
└── README.md               # This file
```

## Conventions

- Date all entries (YYYY-MM-DD)
- Link to relevant source files or external URLs where applicable
- Tag decisions as OPEN, DECIDED, or REVISIT
